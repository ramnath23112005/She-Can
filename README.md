# She Can Foundation

> Empowering youth through education, opportunities, and digital initiatives.

A full-stack NGO website built with **React + Vite** (frontend) and **Node.js + Express + MongoDB** (backend). Features glassmorphism UI, dark/light mode, animated scroll-reveal sections, a contact/volunteer form with email notifications, newsletter subscription, Excel-based offline record-keeping, and an admin dashboard.

**Live site:** [she-can-chi.vercel.app](https://she-can-chi.vercel.app)  
**Live API:** [she-can-oxk3.onrender.com](https://she-can-oxk3.onrender.com)

## Quick Start

### Prerequisites

- Node.js v18+
- MongoDB Atlas account (free tier)

### Backend

```bash
cd she-can-foundation/backend
npm install
# Create a .env file (see backend/.env.example)
npm run dev          # http://localhost:5000
```

### Frontend

```bash
cd she-can-foundation/frontend
npm install
npm run dev          # http://localhost:5173
```

> Update API URLs in `Contact.jsx` and `Admin.jsx` to `http://localhost:5000` for local development.

## Features

- **Modern Hero Section** — Gradient shapes, badge, stats counter, inspiring tagline
- **Glassmorphism UI** — Frosted glass cards with backdrop blur
- **Dark / Light Mode** — Persistent theme toggle with smooth transitions
- **Scroll Animations** — AOS library for fade/slide-up reveal effects
- **Responsive Design** — Fully mobile-optimized with hamburger menu
- **Contact / Volunteer Form** — Client + server validation, success popup
- **Email Notifications** — Admin alerts on form submission, welcome emails on newsletter signup
- **Newsletter Subscription** — Email capture with MongoDB + Excel recording
- **Excel Recording** — All submissions written to `.xlsx` files via ExcelJS
- **Admin Dashboard** — `/admin` route displaying all submissions as styled cards
- **Loading Animation** — Branded full-screen intro with progress bar

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/contact` | Submit contact/volunteer form |
| `GET` | `/api/submissions` | Fetch all submissions (admin) |
| `POST` | `/api/newsletter` | Subscribe to newsletter |
| `GET` | `/api/newsletter` | Fetch all subscribers (admin) |

## Deployment

- **Frontend:** Vercel — set root directory to `she-can-foundation/frontend`, framework preset `Vite`
- **Backend:** Render — set root directory to `she-can-foundation/backend`, start command `node server.js`
- **Database:** MongoDB Atlas — add `0.0.0.0/0` to Network Access for Render connections
